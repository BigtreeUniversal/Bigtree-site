BUHS — Horloge + Radio + Vidéo + News

Horloge client (bague autonome) + vidéo YouTube + playlist (radio) + index newsletter (ouverture en nouvel onglet).

Horloge (client)
bague autonome • phase/latitudes pilotées par la page serveur
Vidéo YouTube
lecture dans une fenêtre dédiée
Radio (playlist YouTube)
joue en continu, pendant que vous lisez la newsletter
Conseil : l’autoplay audio est parfois bloqué ; lancez la lecture manuellement si besoin.
Newsletter (index)
les clics ouvrent dans un nouvel onglet pour ne pas couper la musique
Ouvrir l’index dans un nouvel onglet ↗
Astuce : forcer l’ouverture de tous les liens de la newsletter dans un nouvel onglet Ajoutez ce script dans votre newsletter/index.html (avant </body>) : 
<script>
  document.addEventListener('DOMContentLoaded', function () {
    document.querySelectorAll('a[href]').forEach(a => {
      a.setAttribute('target','_blank');
      a.setAttribute('rel','noopener');
    });
  });
</script>
