I. Introduction

Bienvenue dans l'écosystème BUHS. Le module Radar Clock Fusion v1.0 est une application web progressive (PWA) unique fusionnant deux piliers :

  • Modèle Prédictif : Basé sur l'horloge UTC internationale.
  • Modèle Radar : Observation brute en temps réel sans filtres satellites.

II. Architecture et Mécanique

Le système repose sur une projection azimutale équidistante avec le Nord au centre.

  • Aiguille Jaune (Soleil) : Définit l'heure via la position du soleil (cycle de 24h) calée sur l'UTC.
  • Aiguille Bleue (Lune) : Basée sur le cycle synodique de 29,53 jours pour une représentation en longitude.
  • Aiguille Rouge (Séparateur) : Visualise la lumière électromagnétique (jour/nuit).

III. Innovation : La Bague Azimutale

Cette bague rotative interactive permet :

  • Le réglage manuel du fuseau horaire pour aligner le système sur votre position locale.
  • Une synchronisation personnelle entre l'heure UTC et votre réalité géographique.

IV. Technologie Stratum 1

Le cœur du système bat à Zurich, en Suisse, via le serveur BUHS Heartbeat Azurite.

  • Précision absolue via des recherches sur les horloges atomiques.
  • Protocole Syncro-link propriétaire pour la synchronisation des modules.

V. Installation (PWA)

Pour une immersion totale sans barre d'adresse :

  • iOS : Partage > "Sur l'écran d'accueil".
